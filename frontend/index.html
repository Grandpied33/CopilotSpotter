Bienvenue dans l'ALPHA de SpotterCopilot

Bonjour à tous,

Merci de tester l'Alpha de SpotterCopilot. Cet assistant IA a été conçu dans le but de vous proposer des séances de sport à partir de vos envies et de votre forme. L'IA est censée retenir vos performances que vous pouvez lui donner afin que, sur les prochaines séances, elle optimise vos poids.

Le but de ce test est de :

  • Tester les réponses de l'IA
  • Vérifier qu'elle répond bien à vos attentes
  • Qu'elle s'adapte par rapport aux poids que vous donnez
  • Qu'elle ne boucle pas sur les mêmes exercices

Ce test ne porte pas sur :

  • L'interface
  • Les bugs graphiques
  • La performance de l'IA

Si vous trouvez un bug, n'hésitez pas à ajouter un commentaire et à m'envoyer les chats qui posent problème.

Merci à vous.
Hugo
